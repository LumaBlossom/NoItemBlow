## Features

- **Explosion Protection**: Prevents items from being destroyed by explosions (can be toggled).
- **Fire Control**: Toggle whether items burn in fire.
- **Lava Control**: Toggle whether items burn in lava.
- **In‑Game Configuration**: Simple text‑based UI, visible **only to you**.
- **Language Selection**: Switch the datapack language in‑game.

## Supported Versions

- **Minecraft Java Edition 1.21 and above**

## Installation

1. Download the datapack.
2. Open your world folder.
3. Place the datapack `.zip` into:
   ```
   yourworld/datapacks
   ```
4. Start the world.

## First‑Time Setup

After installing the datapack:

1. Run:

   ```
   /reload
   ```

   **OR** disable and re‑enable the datapack.

2. ⚠️ **Warning**: The **first `/reload` message is shown to all players**.

3. Switch to **Creative Mode**.

4. Click the **`/trigger nibconfig`** button shown in chat
   **OR** type it manually:

   ```
   /trigger nibconfig
   ```

5. The configuration UI will open.

## Configuration Options

Using the in‑game UI, you can:

- Toggle the no item explosion feature **ON / OFF**
- Enable or disable item burning in:
  - **Fire**
  - **Lava**
- Change the **language**

## Notes

- Configuration requires **Creative Mode**.
- Changes apply instantly after selection.
- Designed to be lightweight and server‑friendly.
